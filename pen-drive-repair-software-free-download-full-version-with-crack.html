Fix Pen Drive Data Not Showing Issue

Fix 1: Pen Drive Data Not Showing

If your pen drive is detected and shows the occupying space but not showing data and files, it is probably due to virus infection and makes all data on the pen drive hidden. If you can’t access the data in the pen drive, you may try these methods to solve USB pen drive data now showing issue.

1. Try the first move, connect the pen drive with PC, and click Start -> Control Panel -> Appearance and Personalization -> File Explorer Options -> Show hidden files and folders.

2. The second action you can try is to unhide data on the pen drive by using CMD. You can connect the pen drive to PC, then click Windows + R keys on the keyboard at the same time to open RUN windows. Next type cmd in the box to open Windows Command Prompt window. After that, type diskpart and hit Enter. Finally input this command line: attrib -h -r -s /s /d E:*.*, and hit Enter. Replace “E” with the exact drive letter of your pen drive.

If these two methods don’t work, then you have to recur to the best data recovery software and follow the 3 steps guide above to easily recover data from pen drive for free.

Fix 2: Pen Drive Detected but not Opening

If the pen drive data not showing issue is caused by pen drive detected but not accessible, you may try these measures.

1. Change a drive letter for the pen drive. Connect the pen drive to PC -> Click Windows + R keys -> Type diskmgmt.msc -> Right click the target pen drive -> Change Drive Letter and Paths -> Assign a new drive letter to the pen drive.

2. Run Windows CHKDSK utility to check and repair disk errors. Press Windows + R keys -> Input cmd -> type the command line: chkdsk e: /f /r (“e” refers to drive letter of the pen drive) -> Hit Enter.

Fix 3: Pen Drive Detected but not Visible

The pen drive data cannot be accessed. And the pen drive is detected but doesn’t appear. It may be due to an outdated or incompatible device driver. You can update or reinstall the driver for the pen drive to have a try.

  • Right click This PC and choose
  • Click Device Manager from the left column.

  • After the USB driver is updated or reinstalled, you can reboot your computer and reconnect the pen drive to the PC to check if the pen drive can be accessed now.
